日本收據助手（Japan Receipt Helper）是一款專為赴日旅客設計的旅遊記帳與消費管理工具。

透過 AI 辨識技術，您只需一鍵拍下日本商店收據，App 便能自動提取店名、品項金額與消費類型，並即時轉換為中文資訊，讓複雜的日文收據與海外消費紀錄變得一目了然。

核心功能： * AI 收據辨識：隨手拍下日本當地收據，自動解析店名、即時換算偏好幣別，省去繁瑣的手動輸入，輕鬆完成海外記帳。

附加地理對照（輔助功能）：本服務會嘗試根據收據上的地址文字，在後端進行地理資訊對照。若成功比對，系統會貼心地在畫面上輔助標記，作為您回顧消費地點的額外參考。

隱私守護：本 App 不請求、亦不追蹤您的實時 GPS 位置。所有資訊均基於您上傳的收據文字解析或手動輸入，確保個人隱私絕對安全。

為什麼選擇日本收據助手？
去日本旅行時，總是拿了一堆紙本收據、記不清自己把錢花去哪，甚至看不懂日文店名與明細嗎？這款 App 幫助您將「複雜收據」轉化為「清晰的記帳帳本」。無論是那霸拉麵店的宵夜、還是國際通藥妝店的大血拼，通通幫您整理得清清楚楚、有條不紊！
